Test API

API Documentation

Endpoint

GET /text-to-speech?text=<your_text>

Converts text to speech and returns base64-encoded MP3 audio.

Parameters

  • text: Required. The text to convert (max 1000 characters).

Response (JSON)

{
  "status": "success",
  "audio_base64": "<base64_string>",
  "audio_format": "mp3"
}

On error: {"error": "<message>"}

Example Usage

curl "https://tts-api-liart.vercel.app/text-to-speech?text=Hello%20World"
