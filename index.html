REST API Documentation

This API provides endpoints for text-to-speech conversion and health checks.

Endpoint: /generate-audio

  • Method: POST
  • Description: Converts text to MP3 audio using ElevenLabs.
  • Request Body (JSON): 
    {
  "text": "Your text here",  // Required, max 1000 chars
  "model_id": "eleven_v3",   // Optional, default: eleven_v3
  "speed": 1.0               // Optional, default: 1.0
}
  • Response: Binary MP3 file (audio/mpeg) on success. JSON error on failure (e.g., {"error": "Text too long"}).
  • Status Codes: 200 (Success), 400 (Bad Request), 500 (Server Error).
  • Example Requests:

    cURL

    curl -X POST https://your-project.vercel.app/generate-audio \
-H "Content-Type: application/json" \
-d '{"text": "Hello, this is a test."}' \
--output output.mp3

    Python

    import requests

url = "https://your-project.vercel.app/generate-audio"
data = {"text": "Hello, this is a test."}
headers = {"Content-Type": "application/json"}

response = requests.post(url, json=data)
if response.status_code == 200:
    with open("output.mp3", "wb") as f:
        f.write(response.content)
    print("MP3 downloaded successfully")
else:
    print(f"Error: {response.json()['error']}")

    JavaScript (Fetch)

    async function generateAudio() {
    const response = await fetch('https://your-project.vercel.app/generate-audio', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ text: 'Hello, this is a test.' })
    });
    if (!response.ok) {
        const error = await response.json();
        throw new Error(error.error);
    }
    const blob = await response.blob();
    const url = window.URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'output.mp3';
    document.body.appendChild(a);
    a.click();
    window.URL.revokeObjectURL(url);
    document.body.removeChild(a);
}
generateAudio();

Endpoint: /health

  • Method: GET
  • Description: Checks API health.
  • Response: JSON e.g., {"status": "healthy", "timestamp": "2025-09-13 08:36:00 WIB"}
  • Status Codes: 200 (Success).
  • Example Requests:

    cURL

    curl https://your-project.vercel.app/health

    Python

    import requests

response = requests.get("https://your-project.vercel.app/health")
print(response.json())

    JavaScript (Fetch)

    fetch('https://your-project.vercel.app/health')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));
